个人始终坚信,所有不以实际入侵为目的的安全研究都是耍流氓 [ 目标明确 ]

没有成功付诸实践的 ‘假如’, 也都只是自己一厢情愿的意淫 [ 重在实践 ]

与其浪费大量的时间随波逐流,不如静下心来,从0到1,人云亦云基本是不会什么有出路的,我想要的只是相对本质,相信你也一样 [ 返璞归真,非闭门造车 ]

所有的复杂都是简单的堆积  ”根基不牢,地动山摇” [ 不积跬步,无以至千里 ]

nothing,doing …… [ 少说多做 ]

不用太在意别人说什么,做好你自己,并时刻清楚你自己想要的到底是什么即可 [ 足够专注 ]

把会学习变成一种习惯,并好好把它坚持下去,你会受益终生 [ 不停刷新自己的知识层面 ]

善于站在前辈们的肩膀上,在现有的优秀经验基础上再根据自己的实际需求不断改进,你会走的更稳 [ 善于总结 ]

其实,并不一定非要自己有多么高的智商,始终坚信,足够的勤奋好学,会弥补这一切,只要你愿意,这些都不是问题 [ 足够自信 ]

最后,再跟大家分享一句很久之前在一个前辈的博客上看到的一句话,”我所说的我都做过“,至今对我触动依然非常大,希望能跟大家一起共勉…

放弃那些并不能给你带来什么的 虚荣 ,相信你会活的更洒脱,也会得到更多实实在的东西….

要掌握的东西还很多,后面的路也还很长,希望我们都能一直认真努力地坚持一起走下去



博客的初衷仅仅是希望能给大家提供一个相对靠谱的平台,方便大家一起交流进步

绝不是要怂恿大家利用所学知识去进行任何形式的犯罪,能坚持把文章看下去的,相信都是成年人了,有些底线,大家应该比我更清楚
